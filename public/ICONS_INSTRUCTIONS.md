# Instruções para Criar Ícones e Imagens

## Ícones Necessários

Você precisará criar os seguintes ícones para o site. Você pode usar ferramentas online como:

- **Favicon.io** (https://favicon.io/)
- **RealFaviconGenerator** (https://realfavicongenerator.net/)
- **Canva** (https://canva.com/)

### 1. Ícone Principal (512x512)

- **Arquivo**: `public/icon-512.png`
- **Tamanho**: 512x512 pixels
- **Design**: Fundo azul marinho (#1a365d) com símbolo de igreja branco no centro
- **Formato**: PNG com transparência

### 2. Ícone Médio (192x192)

- **Arquivo**: `public/icon-192.png`
- **Tamanho**: 192x192 pixels
- **Design**: Mesmo design do ícone 512x512
- **Formato**: PNG com transparência

### 3. Favicon (32x32 e 16x16)

- **Arquivos**: `public/favicon-32x32.png` e `public/favicon-16x16.png`
- **Tamanhos**: 32x32 e 16x16 pixels
- **Design**: Versão simplificada do ícone principal
- **Formato**: PNG

### 4. Favicon.ico

- **Arquivo**: `public/favicon.ico`
- **Formato**: ICO (multi-size: 16x16, 32x32, 48x48)
- **Pode ser gerado a partir do PNG usando favicon.io**

### 5. Apple Touch Icon

- **Arquivo**: `public/apple-touch-icon.png`
- **Tamanho**: 180x180 pixels
- **Design**: Mesmo design do ícone principal
- **Formato**: PNG

### 6. Imagem Open Graph

- **Arquivo**: `public/og-image.jpg`
- **Tamanho**: 1200x630 pixels
- **Design**:
  - Fundo com gradiente azul marinho (#1a365d)
  - Texto "ADMAC - Igreja Missionária" em fonte elegante
  - Subtítulo "Comunidade de Fé e Adoração"
  - Ícone da igreja
  - Elementos decorativos dourados (#ecc94b)
- **Formato**: JPG otimizado (< 300KB)

## Ferramentas Recomendadas

1. **Para criar ícones simples**: https://favicon.io/favicon-generator/
2. **Para criar todos os favicons de uma vez**: https://realfavicongenerator.net/
3. **Para criar a imagem Open Graph**: Canva ou Figma

## Solução Temporária

Enquanto você não cria os ícones personalizados, você pode:

1. Usar o logo existente da igreja (se houver)
2. Usar geradores online para criar ícones básicos
3. O site funcionará mesmo sem esses arquivos, mas terá avisos no console
